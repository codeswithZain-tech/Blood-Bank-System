# OOP Project Hospital Blood Bank Management System(C++)

## Overview
A comprehensive C++ application for managing blood bank operations in a hospital setting with role-based access control.

## User Roles
- **Admin**: Full system control, user management
- **Staff**: Day-to-day operations, inventory management
- **Donor**: View profile and appointments

## Features
- Blood inventory management (Add/Subtract units)
- Donor registration and management
- Patient request fulfillment
- Appointment scheduling
- Eligibility checking (age 18-65)
- CSV file-based data persistence

## Data Files
- donors.txt - Donor information
- patients.txt - Patient records
- blood_inventory.txt - Blood stock
- appointments.txt - Donation schedules
- users.txt - System users

## Built With
- C++ (No external libraries)
- File I/O for data persistence
- OOP principles (Inheritance, Polymorphism)
